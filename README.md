# API сервис для публикации обучающих курсов
## Запуск проекта
## Шаг 1
**Заполните файл .env.example и переименуйте его в .env**
## Шаг 2
**Запустите контейнеры командой**
```commandline
docker-compose up
```
Шаг 3
**Что бы посмотреть доступные эндпоинты, выполните запрос на адрес**
```commandline
/swagger/
```
