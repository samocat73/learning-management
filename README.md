# API сервис для публикации обучающих курсов
## Запуск проекта
## Шаг 1
**Заполните файл .env.example и переименуйте его в .env**
## Шаг 2
**Запустите контейнеры командой**
```commandline
docker-compose up
```
Шаг 3
**Что бы посмотреть доступные эндпоинты, выполните запрос на адрес**
```commandline
/swagger/
```
## Размещение проекта на виртуальной машине
## Шаг 1
**Настройте GitHub Secrets, в нем должны быть:**
**ENV_FILE #Сюда нужно поместить заполненные данные из шаблона .env.example** 
**DOCKER_HUB_USERNAME**
**SERVER_IP**
**SSH_KEY**
**SSH_USER**
## Шаг 2
**На виртуальную машину установите Git и Docker**
**Настройте файрвол**
